# Sky Flux Skills

Agent skills for Claude Code by [Sky Flux](https://github.com/sky-flux).

## Skills

| Skill | Description |
|-------|-------------|
| [elysia](./skills/elysia/) | One-stop ElysiaJS skills covering full website, all blog posts, and all GitHub repository insights |

## Usage

Add to your Claude Code configuration:

```json
{
  "skills": ["sky-flux/skills"]
}
```
